# CRUD PROJECT BY USING PYTHON-FASTAPI-SQLALCHEMY

This is CRUD project. It's means create,read,update and delete.

## Project Description

```bash
git clone "git@github.com:IshtiAhmedTiham/crud.git"
```


## Installation
FASTPI ->
```bash
pip install "fastapi[standard]"
``` 

SQLALCHEMY -> 
```bash
pip install sqlalchemy
``` 


## Usage

```python
from fastapi import FastAPI

# make a blueprint
app = Fastapi()
```

## Run 

```bash
fastapi dev main.py
``` 
Another way to run->
```bash
python3 -m uvicorn main:app --reload
```
